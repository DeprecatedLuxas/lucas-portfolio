# lesetid

`/leːsətiːd/`, Norwegian: "lesetid"

> This is a monorepo containing the following projects:
>
> * [lesetid](https://github.com/luxass/lesetid/blob/main/packages/lesetid)
> * [playground](https://github.com/luxass/lesetid/blob/main/www)
> * [remark-lesetid](https://github.com/luxass/lesetid/blob/main/packages/remark-lesetid)
> * [vscode-reading-time](https://github.com/luxass/lesetid/blob/main/packages/vscode)

## 💻 Development

* Clone this repository
* Enable [Corepack](https://github.com/nodejs/corepack) using `corepack enable` (use `npm i -g corepack` for Node.js < 16.10)
* Install dependencies using `pnpm install`
* Run tests using `pnpm dev`
